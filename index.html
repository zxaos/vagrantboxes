Ubuntu Server 13.04 Vagrant Base Box

This is a basebox for Vagrant. It is using Ubuntu Server 13.04 x64, with

  • 360MB memory by default
  • 12MB VRAM by default
  • 40GB Auto-resizing HDD
  • No USB
  • No audio
  • No serial
  • No X11
  • A minimal virtual machine ubuntu install
  • Canonical Landscape removed
  • Virtualbox Guest Utilities
  • Chef (10.12.0) and Puppet (2.7.18) installed from Ubuntu repos
  • Default Vagrant SSH support

It was last fully updated on April 29, 2013. However, since the VirtualBox Guest Utilties were installed via DKMS, there shouldn't be any problems running apt-get upgrade.

You can download the most recent version from http://zxaos.github.io/vagrantboxes/raringServer64.box
